Sobre o IMC

O Índice de Massa Corporal (IMC) é uma medida utilizada para avaliar se o peso de uma pessoa está adequado em relação à sua altura. Ele é calculado dividindo o peso (em quilos) pela altura ao quadrado (em metros).

Atenção:

O IMC é apenas um indicador e não substitui uma avaliação médica. Outros fatores, como idade, composição corporal e condições de saúde, também devem ser considerados.

Acima de 40,0 // Obesidade grau III

Entre 35,0 e 39,9 // Obesidade grau II

Entre 30,0 e 34,9 // Obesidade grau I

Entre 25,0 e 29,9 // Sobrepeso

Entre 18,6 e 24,9 // Normal

18,5 ou menos // Abaixo do normal

OBS: Utilize "." ou "," para informar a altura
